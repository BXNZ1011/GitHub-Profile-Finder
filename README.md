# GitHub Profile Finder

A modern GitHub Profile Finder built with HTML, CSS, and JavaScript.

## Features

- Search any GitHub username
- View avatar, bio, followers, following, and repositories
- Responsive design
- Glassmorphism UI
- Dark/Light mode
- Uses the GitHub REST API

## Technologies

- HTML5
- CSS3
- JavaScript
- GitHub API

## How to Run

1. Download the project.
2. Open `index.html` in your browser.
3. Enter a GitHub username.
4. Click **Search**.
